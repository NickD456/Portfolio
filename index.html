Milksong

Engine: Raylib

Language: C++

Watch our beta trailer!

Milksong is a 2d action platformer built in C++ using raylib as its graphics library. The game features 4 distinct levels with multiple upgrades for the player to obtain. They will face off against both melee and ranged enemies while traversing the map collecting power ups such as extra health, a double jump and a ranged attack.

Zombie Cafe

Engine: Unity

Language: C#

Zombie Cafe is a game I made with a team of 2 others for my school project. I worked as the lead programmer for the game. The game is set in a zombie apocalypse where during the day you are tasked with running a cafe. At night you need to defend the cafe from the incoming zombies. The game has a fully functional recruiting system where you are able to recruit customers during the day who help defend the cafe when night comes. Recruited NPCS are also able to be sent on looting quests to find money and other supplies.

Github link for the game

VR Museum

Engine: Unity

Language: C#

I worked on this project in collaboration with the engineering department at SUNY Polytechnic Institute.
I created a program where students were able to interact with and manipulate the parts they are working with in a virtual environment. Using programs like Revit and SolidWorks I translated CAD designs into models suitable for the Unity game engine
